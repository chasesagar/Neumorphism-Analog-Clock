# Neumorphism-Analog-Clock
How to make a Neumorphism analog clock using only simple HTML, CSS, and JavaScript programming code.See [demo](#demo)

## About

This is a real-time clock, which means that the clock sets its own time according to the time of your device. The role of JavaScript programming code is the most important in making it. It is structured using HTML code only. The CSS code helped to design it and sort it out in the form of Neumorphism. This watch is enabled using JavaScript programming code. JavaScript programming has a role to play this clock according to the time of your device.
